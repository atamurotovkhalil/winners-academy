

#  Winners Academy – Education Platform

Winners Academy is a modern educational web platform designed to deliver high-quality learning experiences through an intuitive, responsive, and performance-optimized interface. The platform focuses on content delivery, user engagement, and scalability.

---

##  Features

*  Educational content platform for courses and learning materials
*  Modern and responsive UI for all devices
*  Smooth client-side routing
*  Rich text editing for content creation
*  PDF viewing support
*  Fast build and optimized performance
*  Scalable and maintainable frontend architecture

---

##  Tech Stack

### Frontend

* **React 18**
* **TypeScript**
* **Vite**
* **React Router**
* **Tailwind CSS**
* **Zustand** (state management)

### UI & UX

* Tailwind CSS + Tailwind Merge
* Radix UI
* Headless UI
* Heroicons & Lucide Icons
* Embla Carousel
* AOS (Animate on Scroll)

### Editors & Media

* TipTap Editor (rich text editing)
* React Quill
* PDF.js / React PDF

### Development Tools

* ESLint
* TypeScript
* Vite Plugin React


## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/atamurotovkhalil/winners-academy.git
cd winners-academy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Preview production build

```bash
npm run preview
```

## Scripts

| Command           | Description              |
